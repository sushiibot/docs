# sushii-docs

This is the documentation for [sushii-2](https://github.com/sushiibot/sushii-2).

## Setup

sushii-docs uses [mdBook](https://rust-lang.github.io/mdBook/)

```bash
# Install mdbook
cargo install mdbook

# Render book
mdbook build

# For development
mdbook watch
```
